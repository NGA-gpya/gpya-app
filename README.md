# GPYA - Aplicación de Documentación Legal

Aplicación móvil multiplataforma desarrollada en **Flutter** para **Padilla y Aguilar**. Permite la visualización y descarga de documentación legal clasificada por normativas.

## 🚀 Funcionalidades Principales

- **Centro de Información:** Visualización de documentos legales desde Supabase.
- **Búsqueda Avanzada:** Filtrado por nombre y palabras clave.
- **Categorización:** Clasificación por Normas de Seguridad, Salud, Organización y Específicas.
- **Gestión de Descargas:** Descarga de archivos PDF y visualización local.
- **Tema Dinámico:** Soporte para modo claro, oscuro y del sistema.

## 🛠️ Requisitos

- Flutter SDK (^3.9.0)
- Configuración de Supabase (URL y Anon Key en archivo `.env`)

## 📦 Construcción (Build)

Para generar el instalador de producción:

```bash
flutter build apk --release
```

---

## 🛠️ Especificaciones Técnicas

- **Versión actual:** 1.0.0
- **Peso del APK:** 50.9 MB
- **Requisitos:** Android 5.0 (Lollipop) o superior.
- **Tecnología:** Flutter + Supabase (Real-time Cloud Sync).

## 📖 ¿Cómo funciona?

1.  **Instala:** Descarga el APK, habilita "Fuentes desconocidas" e instala.
2.  **Busca:** Localiza documentos por nombre o categoría (Seguridad, Salud, etc.).
3.  **Descarga:** Guarda los archivos localmente para acceso offline inmediato.
